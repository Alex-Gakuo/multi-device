Develop apps for Chrome Mobile browsers

Google provides three ways to run web content on mobile devices:

  • Using Chrome Mobile for devices running Android 4.0 and later, and iOS.
  • With the default Android browser, for devices running versions of Android earlier than 4.0 (Ice Cream Sandwich).
  • Within WebView on a native Android application.

Chrome Mobile browser is the base WebKit browser implemented by both Google Chrome for Android and Apple Mobile Safari. There are differences in the implementations of each. See the following pages:

  • Chrome for Android
  • Chrome for iOS

Getting started
